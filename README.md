This is a personal product management case study focused on identifying student productivity challenges and supporting solution definition using structured PM practices.

## My Responsibilities

## Target Persona (Summary)
Primary Persona:
An undergraduate student managing multiple academic deadlines across different platforms, often feeling overwhelmed and missing tasks due to poor prioritization.
